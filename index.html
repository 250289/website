Cross plataform, open source .NET framework

Mono é uma plataforma de softwware construída para permitir que desenvolvedores possam criar facilmente aplicações cross plataforma.

Patrociada pela Xamarin, Mono é uma implementação open source do Microsoft .NET Framework baseado nos padrões ECMA no C# e na Common Language Runtime. Com o crescimento de uma família de soluções e um entusiasmada comunidade contribuidora é o que tem ajudado a posicionar o Mono com a principal escolha para o desenvolvimento de aplicações cross plataforma.

Projeto Mono Brasil
